## Shop  
简介：由于对商城感兴趣而开发的一个基于Web端的网上商城，并通过该项目对当时刚学习的Java框架以及前端框架进行实践。   

### 功能    
功能较为完整，包括商品模块，购物车模块，订单模块，短信/邮件通知等模块，并有相应的管理系统进行管理。

### 主要技术    
Jsp,Spring,Struts2,Hibernate  
前端：JQuery,EasyUI  
数据库：MySQL  
版本控制：git
